Neste exemplo podem ver-se as diferenças relativas ao modo
de activação do objecto remoto.

São fornecidos 3 ficheiros de configuração:
Server.exe.singleton
Server.exe.singlecall
Server.exe.activated

Para colocar o servidor a funcionar num dos modos
basta copiar um destes ficheiros de configuração
para Server.exe.config

No cliente deve ser feita a mesma coisa (para o
mesmo modo de activação) em relação ao ficheiro
Client.exe.config
